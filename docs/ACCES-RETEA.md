# Acces de pe telefon (același Wi‑Fi)

1. **Pe PC:** `npm run dev`
2. **Află IP-ul PC-ului:** în CMD rulează `ipconfig` → notează **Adresa IPv4** (ex: 192.168.1.5 sau 10.5.0.2)
3. **Pe telefon/tabletă:** în browser deschide **http://[acel-IP]:3000** (ex: `http://10.5.0.2:3000`) – obligatoriu cu **http://** în față
4. **Dacă nu se încarcă:** click dreapta pe `scripts\add-firewall-rule.bat` → **Rulează ca administrator** → apoi repetă pasul 3
